HTTP请求与响应

发表于 2018-02-21 | 分类于 HTTP

Server + Client + HTTP

  • 浏览器负责发起请求
  • 服务器在 80 端口接收请求
  • 服务器负责返回内容（响应）
  • 浏览器负责下载响应内容

HTTP 的作用就是指导浏览器和服务器如何进行沟通。
