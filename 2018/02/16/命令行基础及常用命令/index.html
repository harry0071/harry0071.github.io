命令行(CLI)基础及常用命令

发表于 2018-02-16 | 分类于 命令行

介绍了cmd，gitBash命令行(CLI)的一些基础知识，以及常用的一些命令

参考资料: 饥人谷

基本概述

~ / . .. $ 的意思

  1. ~/desktop : 其中的 ~ 表示电脑的username，所以这句命令等同于stage/desktop
  2. . : 当前目录
  3. .. : 上级目录，cd ..表示进入上级目录

基础命令

  1. cd ~/Desktop 进入桌面
    1.1 cmd直接cd desktop
  2. mkdir demo-1 创建目录，这时你可以切到桌面，看到 demo-1 目录
  3. rm -rf demo-1 删除目录【-r表示递归，循环删除直至删光，-f表示强制，不提示yes/no】
  4. touch 1.txt 创建文件
    4.1 cmd用echo x > xxx.txt
  5. mv 1.txt 2.txt 这样我们就把 1.txt 移到 2.txt 了，也就是重命名
  6. cp js/main.js vendor,将js目录下的main.js复制到vendor目录下
    6.1 cmd用copy，比如copy C:\Users\stage\Desktop\hello\目录1>1.txt C:\Users\stage\Desktop\hello，将目录1里的1.txt复制到hello目录内

cmd命令行

进入目录

如果要进入E:\aaa\bbb\ccc
首先要C:\Users\stage>e:进入E盘
然后在进入目录cd E:\aaa\bbb\ccc
【总之，不能直接cd 路径，必须先进入相对应的磁盘，才能cd】

树形列表：

  • tree 路径：只展示目录
  • tree /f 路径:深度展示
  • tree /f 路径> tree.txt：把生成的文件目录树形结构写入到tree.txt文件中

批量合并文件

copy /b E:\aaa\bbb\*.ts E:\aaa\bbb\together.ts

批量删除文件

del E:\aaa\bbb\000*.ts
