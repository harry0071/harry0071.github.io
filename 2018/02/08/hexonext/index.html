hexo的next主题个性化教程:打造炫酷网站

发表于 2018-02-08 | 分类于 hexo

看到有些next主题的网站很炫酷，那么是怎么配置的呢？接下来我会讲一讲如何实现一些炫酷的效果

主要有以下32种：


  1. 在右上角或者左上角实现fork me on github

  2. 添加RSS

  3. 添加动态背景

  4. 实现点击出现桃心效果

  5. 修改文章内链接文本样式

  6. 修改文章底部的那个带#号的标签

  7. 在每篇文章末尾统一添加“本文结束”标记

  8. 修改作者头像并旋转

  9. 博文压缩

  10. 修改代码块自定义样式

  11. 侧边栏社交小图标设置

  12. 主页文章添加阴影效果

  13. 在网站底部加上访问量

  14. 添加热度

  15. 网站底部字数统计

  16. 添加 README.md 文件

  17. 设置网站的图标Favicon

  18. 实现统计功能

  19. 添加顶部加载条

  20. 在文章底部增加版权信息

  21. 添加网易云跟帖(跟帖关闭，已失效，改为来必力)

  22. 隐藏网页底部powered By Hexo / 强力驱动

  23. 修改网页底部的桃心

  24. 文章加密访问

  25. 添加jiathis分享

  26. 博文置顶

  27. 修改字体大小

  28. 修改打赏字体不闪动

  29. 自定义鼠标样式

  30. 为博客加上萌萌的宠物

  31. DaoVoice 在线联系

  32. 点击爆炸效果
