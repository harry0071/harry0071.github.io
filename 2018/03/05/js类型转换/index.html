内存图

  • JS 引擎将内存分为代码区和数据区

我们只研究数据区

  • 数据区又分为 Stack区（栈内存） 和 Heap区（堆内存）
  • 简单类型的数据直接存在 Stack栈内存 里
  • 复杂类型(object)把地址存到 Stack栈内存 区，把地址对应的数据存到 Heap堆内存 区

o2 = o;并不是修改了o2在Heap的数据，而是将o2的地址修改成o的地址，因此2访问o对应的数据
